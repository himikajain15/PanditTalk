## Quick Start

**Choose one:**

**For Command Prompt:**
```
START.bat
```

**For PowerShell:**
```
.\START.ps1
```

This opens:
- Backend (http://localhost:8000)
- Pandit App (http://localhost:8081)
- User App (http://localhost:8082)
- Admin Panel (http://localhost:8000/admin)

## Run on Phone (Wireless)

**One-time setup:** See `WIRELESS_SETUP.txt` for pairing instructions

**Daily use:**
- User App: `RUN_USER_APP.bat` (Backend + App launch together)
- Pandit App: `RUN_PANDIT_APP.bat` (Backend + App launch together)
